### Важные правила для всех

- Свою задачу делаешь в отдельной ветке от `dev`
  - Ветка для новой фичи: `feat/описание`
  - Ветка для исправления: `fix/описание`
- Вливаешь в `dev` только через **Pull Request**
- Перед началом работы:
```bash
  git checkout dev
  git pull origin dev
```
- Если что-то непонятно или заблокировано — сразу пишешь в общий чат
